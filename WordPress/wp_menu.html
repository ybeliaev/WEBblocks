Регистрация меню в своей теме

Нужно подвесить хук на событие after_setup_theme с использованием register_nav_menu()

                add_action( 'after_setup_theme', 'action_after_setup' );
                function action_after_setup() {
                    // Действия
                    register_nav_menu( 'top', 'Top Menu' );
                    register_nav_menu( 'footer', 'Footer Menu' );
                }

Теперь в админке Внешний вид - Меню можно создать меню. После создания в админке меню его нужно подключить к вёрстке. В этом нам поможет функция wp_nav_menu()

Вот с подобным меню поработаем ( код нужно в обёртку nav поместить):

В function.php прописываем, убирая обёртку в элемент div

                wp_nav_menu( array(
                    'theme_location'  => 'top',
                    'container'       => null,                     
                    'items_wrap'      => '
    %3$s
',                    
                ) );

Внимание: у WP сложно повесить класс .active, т.к есть свой класс current-menu-item . Поэтому или добавить этот класс в стилях или им заменить класс .active При этом наш js не применяется для выделения пункта - WP позаботится сам.

Добавление миниатюр

Для добавления миниатюр используется функция add_theme_support()

Добавляем add_theme_support( 'post-thumbnails' ); в function action_after_setup(). Заходим в Настройки - Медиафайлы и устанавливаем нужный размер миниатюр по-умолчанию
