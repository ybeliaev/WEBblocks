Теперь сотворим эффект 1. "вылетания" сверху (ось Y) и 2. появления элементов картинки, двух заголовков и абзаца

Для второго эффекта нужно задать разное время задержки, тогда увидим последовательное появление элементов:

            .rocky-dashed {  
                  animation-delay: .6s;
            }

            .header-title {
                animation-delay:.8s;
            }

            .header-subtitle {
                animation-delay: 1s;
            }

            .header-button {
                animation-delay: 1.1s;
            }

Теперь сама анимация( в ней добавить эффект скатывания сверху ):

            @keyframes pop-in {
              0% {
                opacity: 0;    
                transform: translateY(-4rem) scale(.8);
              }
              100% {
                opacity: 1;    
                transform: none;
              }
            }

В результате к каждому элементу кроме собственных классов, описаных выше, нужно добавить общий класс:

            .animate-pop-in {  
              animation: pop-in .6s cubic-bezier(0, 0.9, 0.3, 1.2) forwards;
              opacity: 0; 
            }

Важно указать нулевую прозрачность по-умолчанию. Иначе элементы вначале не будут скрыты.
