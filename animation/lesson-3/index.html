Создаём мигающий и пропадающий знак скрола

Вставляем обычную картинку скрола со свойством bottom: 4vh

            .header-down-arrow {              
              bottom: 4vh;
              left: 0;
              position: absolute;
              right: 0;
              text-align: center;
              z-index: 10;
            }

Создаём правило ( будет эфект появления и поднятия снизу ):

            @keyframes fade-slide-up {
            0% {
              opacity: 0;
              transform: translateY(4rem);
              }
            100% {
              opacity: 1;
              transform: none;
              }
            }

            .header-down-arrow img {
              animation: fade-slide-up 1s 1s ease-out forwards;
              opacity: 0;
            }

Создаём правило эффекта пульсации:

            @keyframes pulse {
              0% {
                opacity: 1;
                transform: none;
              }
              50% {
                opacity: .8;
                transform: scale(.8);
              }
              100% {
                opacity: 1;
                transform: none;
              }
            }
            
            .header-down-arrow {
              animation: pulse 2s 3s ease-out infinite;
              ...
            }

Есть ещё один вариант написания анимаций:

            Правило:
            @keyframes no-transform {
              100% {
                opacity: 1;  
                transform: none;
              }
            }
            И добавляется к элементу. Причём важно добавить свойство, которое будет меняться:
            .header-down-arrow img { 
              animation: no-transform 1s 1s ease-out forwards;
              opacity: 0;  
            ->  transform: translateY(4rem);
            }
