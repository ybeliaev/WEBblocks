Загрузка страницы.

Элементы страницы могут иметь разный вес и при плохой скорости загрузки это исказит анимацию.
Нужно начинать показ страницы только после полной загрузки самой страницы

Для этого зададим всем элементам страницы класс со свойством, значение которого остановит анимацию:

          .js-loading *, 
          .js-loading *:before, 
          .js-loading *:after {
            animation-play-state: paused !important;
          }

Далее просто после загрузки убираем этот класс:

          document.body.className += " js-loading";

          // Listen for when everything has loaded
          window.addEventListener("load", showPage, false);

          function showPage() {
          // Remove the "js-paused" class
          document.body.className = document.body.className.replace("js-loading","");
          }
