Анимируем полоску под ссылкой

Лучше это делать псевдоэлементом

Иногда удобно использовать em. Ним устанавливается зависимость от размера шрифта

          a {
            color: #2F56B0;
            position: relative;
            text-decoration: none;  
            transition: color .4s ease-out; Для изменения цвета шрифта при эффекте hover
          }

          a::after {
            border-radius: 1em; 
            border-top: .1em solid #2F56B0; Собственно сама полоска
              bottom: .14em;
            content: "";
              left: 0;
            position: absolute;
              right: 100%;  
            transition: border-color .4s ease-out;
          }

          a:hover {
            color: #457DFB;
            right: 0;
            text-decoration: none;
          }

          a:hover::after { 
            animation: anchor-underline 2s cubic-bezier(0,.5,0,1) infinite;
            border-color: #457DFB;
          }

Само правило анимации:

          @keyframes anchor-underline {
            0%, 10% {
              left: 0;
              right: 100%;
            }
            40%, 60% {
              left: 0;
              right: 0;
            }
            90%, 100% {
              left: 100%;
              right: 0;
            }
          }

Анимация попроще.

Навесим анимацию на это слово

Как работает left right на абсолютно спозиционированном элементе: left:0 right:100% загонит элемент в левый угол, left:100% right:0 в правый. Свойства left right работают как margin

Собственно "код"

          span.animation::after {
            border-radius: 1em;
            border-top: .1em solid hsl(160, 100%, 70%);
              bottom: .14em;
            content: "";
            position: absolute;
              left: 0;  
              right: 100%;  
            transition: right .4s ease-out;
          }

          span.animation:hover:after { 
            right: 0; 
          }

Похожий приём, но полоска расходится от центра: слово
